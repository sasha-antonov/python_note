В python 3 логические операции(>>, &, |, etc) производятся с числами(int), для того чтобы перевести из байтового представления в число используется команда ord()

x = b'\x01'
x = ord(x)
x = x<<1

Для преобразования int в байты используется комманда to_bytes(x,'y'). Первый аргумент отвечает за количество байт для предаставления числа. Второй аргумент ('big' или 'little') для определения старший байт впереда или младший байт вперед.

x=1
x=x.to_bytes(2,'big')
x=x.to_bytes(3, 'little')
